GeneralLib
----------

This is just a library of generic Algorithms and Data strucures. The intention of this project is entirely academic.

Algorithms
----------

### Implemented
* Huffman encoding

### Not Implemented 
* Djikstra
* Bellman ford
* Prim's Kruskal's


Data Structures
---------------
### Implemented
* RBT
* AVL

### Not Implemented 
* Fibonacci Heap & Generic heap.
* Union find.
